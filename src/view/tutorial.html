FREE TECA

Esse é um breve tutorial informando como funciona o sitema.


O sistema é direcionado para o cadastro de estudantes que desejam fazer empréstimos de livros na biblioteca. Para realizar esse cadastro, é necessário que o estudante clique no botão de cadastro na página inicial do site, logo em seguida ele será redirecionado para uma página na qual precisará preencher os campos vazios com os dados solicitados. Após informar todos os dados, deve-se selecionar a opção confirmar. Se todas as informações estiverem de acordo, o usuário será validado para fazer empréstimos.
