O sistema é direcionado para o cadastro de estudantes que desejam fazer emprestimos de livros na biblioteca. Para realizar esse cadastro, é necessario que o estudante clique no botao de cadastro na pagina inicial do site, logo em seguida ele será redirecionado para uma pagina na qual precisara preencher os campos vazios com os dados solicitados. Apos informar todos os dados, deve-se selecionar a opcao confirmar. Se todas as informacoes estiverem de acordo, o usuario sera validado para fazer emprestimos.
